--------- Aurora Ecommerce ----------

Aurora es mi proyecto de Ecommerce el cual comenzo como un aprendizaje de Bootstrap 5 y termino escalando a un repaso de todo lo aprendido de PHP MVC en POO y MySQL en UTN

-Aurora tiene una logica de negocios ficticia, la cual buscaba mas enfocarse en el desarrollo del CRUD. Los productos se encuentran definidos en la Base de datos
junto con datos de animaciones para evitar el uso de Javascript, el desafio era hacerlo en lo posible solo con PHP nativo.

-En Aurora podremos subscribirnos a un newsletter el cual quiero suplantar la funcion mail con PHPmailer para generar emails personalizados
-Tambien podremos agregar y eliminar productos al carrito de compras y generar un checkout ficticio el cual simula como si la compra se hubiese realizado mostrando los poductos y el costo total

El proximo desafio para aurora es implementar PHPmailer, ampliar la logica de negocios ficticia para llevar mas datos al back y menos al front, tambien implementar Javascript nativo para animaciones
y poder pasar a sus respectivos Frameworks.

y mis otros proyectos a Laravel.
